Install Hugo

January 1, 0001

按照官方文档操作即可，不再赘述。 https://gohugo.io/getting-started/quick-start/

brew install hugo
hugo version
hugo new site blog
cd blog
git init

git submodule add https://github.com/panr/hugo-theme-terminal.git themes/terminal
echo 'theme = "terminal"' >> config.toml

hugo new posts/install_hugo.md

hugo server -D
